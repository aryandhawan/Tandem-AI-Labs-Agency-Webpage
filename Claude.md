# Tandem AI Labs — Website Development Instructions

## Project Overview
This is the agency landing page for Tandem AI Labs — a solo AI consulting agency 
that builds custom RAG systems, AI workflow automation, and custom integrations 
for SMBs and startups. The site is built on a v0 by Vercel template (Next.js + 
Tailwind CSS) and is being repurposed from a SaaS product template into a 
professional agency landing page.

## Who This Site Is For
The primary visitor is a business owner, startup founder, or agency owner who 
found Tandem AI Labs through LinkedIn or direct outreach. The site has ONE job: 
convert a curious visitor into someone who books a free 15-minute business audit 
call or sends an email. Every design and content decision should serve this goal.

## Tech Stack
- Framework: Next.js (v0 template)
- Styling: Tailwind CSS
- Fonts: Keep existing template fonts unless instructed otherwise
- Icons: Keep existing template icon library
- Animations: Keep existing template animations (rotating globe, etc.)
- No new dependencies without explicit instruction

## Design Rules — NON NEGOTIABLE
- Keep ALL existing animations and visual effects exactly as they are
- Keep ALL existing spacing, layout structure, and component architecture
- Keep the existing light theme as the default
- Add a dark mode toggle that switches between light and dark — dark mode 
  should use a clean dark background (#0a0a0a or similar) with white text
- Never break existing responsive behavior
- Never add unnecessary complexity — keep it clean and minimal
- Color accent: keep the existing black/dark accent from the template
- Do NOT add new sections without explicit instruction
- Do NOT remove sections without explicit instruction — wait to be told 
  which sections to remove one at a time

## Content Rules — CRITICAL
- NEVER use fabricated numbers, fake metrics, or made-up statistics
- NEVER use fake testimonials, fake client names, or fake company logos
- NEVER claim certifications, compliance standards (SOC 2, ISO 27001, HIPAA, 
  GDPR), or uptime SLAs that are not verified and real
- ALL content must be honest and accurately represent what Tandem AI Labs 
  actually delivers
- Placeholder content from the original template must be replaced with real 
  content as instructed — never left as-is
